# Selenium Python Pytest Framework

## 🚀 Overview

Enterprise automation testing framework built using:

- Selenium WebDriver
- Python
- Pytest Framework
- Page Object Model (POM)

---

# 🛠️ Features

✅ Page Object Model  
✅ Reusable Utilities  
✅ Cross Browser Testing  
✅ Parallel Execution  
✅ Screenshot Capture  
✅ Logging & Reporting  
✅ Jenkins CI/CD Integration  

---

# 📂 Framework Structure

```txt
tests/
pages/
utilities/
config/
reports/
```

# ▶️ Run Tests

```bash
pytest -v
```
